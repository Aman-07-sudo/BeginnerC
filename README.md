# BeginnerC

A collection of C programs I built while learning C from scratch. Every project here was written manually — no copy pasting, no shortcuts.

## Projects

### 1. [String Reverser](./String_reverser/)
Reverses any string input by the user without using built-in reverse functions.
- Manual character swapping using arrays
- Custom index manipulation

### 2. [Password Verifier](./Psswd_Verifier/)
A command-line password validator that checks multiple criteria before accepting a password.
- Custom string length function (no strlen)
- ASCII character comparisons
- Multiple validation checks with flag variables

## What I'm learning
- How strings work in memory
- Pointers and character arrays
- Writing functions in C
- Loops, conditionals and flag variables
- How to debug in C (the hard way 💀)

## Setup
You need `gcc` installed. To compile any project:
```bash
gcc filename.c -o output
./output
```
